# MVP Scope - SEPETAK

## Target MVP

MVP fokus pada publikasi informasi dan pengelolaan data inti organisasi dengan
admin panel yang mudah dipakai.

## Fitur MVP - Website Publik

- Beranda
- Profil organisasi
- Daftar artikel/berita
- Halaman detail artikel
- Daftar kasus dan advokasi (ringkas)
- Kontak dan formulir pendaftaran anggota

## Fitur MVP - Admin Panel

- Login admin
- Manajemen artikel/berita
- Manajemen halaman statis
- Manajemen anggota (data dasar dan status)
- Manajemen kasus (status dan timeline sederhana)
- Manajemen advokasi (program dan dokumentasi)
- Upload dokumen dan foto

## Fitur Non-MVP (Nanti)

- Donasi online dan payment gateway
- Integrasi WhatsApp atau SMS
- Mobile app
- API publik
- Multi bahasa
- Advanced analytics dan BI

## User Stories (Ringkas)

- Sebagai publik, saya bisa membaca profil organisasi dan berita terbaru
- Sebagai admin, saya bisa menambah dan mengedit artikel
- Sebagai pengurus, saya bisa mencatat kasus dan status penanganannya
- Sebagai admin, saya bisa mengelola data anggota dan status aktif

## Acceptance Criteria

- Admin dapat login dan mengelola konten tanpa error
- Publik dapat mengakses artikel dan profil dengan cepat di mobile
- Data anggota dapat dicari dan difilter
- Kasus dan advokasi memiliki status yang jelas
