# Moodboard (3 Varian)

Dokumen ini merangkum 3 opsi arah visual yang masih satu keluarga dengan identitas “Tani Merah”, tetapi lebih lembut (softer), lebih mudah dibaca jangka panjang, dan tetap punya personality yang kuat.

## Target Karakter

- Professional namun approachable
- Modern dengan sentuhan personal
- Clean layout dengan personality kuat
- Brand voice konsisten: tegas, ringkas, bernada editorial

## Varian A — “Tani Merah Soft” (Direkomendasikan)

Inti: tetap merah sebagai pemimpin hierarki, tapi merahnya dibuat terracotta/muted; latar paper lebih hangat; aksen hijau-sage tipis untuk elemen sekunder agar tidak monoton.

**Palet (token → nilai)**
- Paper: `hsl(42 45% 96%)`
- Ink: `hsl(222 15% 12%)`
- Primary: `hsl(352 62% 45%)`
- Primary-weak: `hsl(352 55% 92%)`
- Secondary (ochre): `hsl(38 70% 48%)`
- Accent (sage): `hsl(152 22% 34%)`

**Rasa visual**
- Ramah di mata (kontras tetap kuat karena ink gelap).
- Aksen hijau hanya untuk “status/utility” (tag, highlight ringan).

**Mood referensi (kata kunci)**
- editorial, zine, poster modern
- clean grid, big headings, subtle color blocks

## Varian B — “Kopi & Kertas”

Inti: primary digeser sedikit ke “earth brown” sebagai nuansa hangat, cocok untuk konten panjang; merah dipakai sebagai accent kuat untuk CTA.

**Palet (token → nilai)**
- Paper: `hsl(40 55% 95%)`
- Ink: `hsl(25 18% 12%)`
- Primary (earth): `hsl(24 45% 34%)`
- Primary-weak: `hsl(24 35% 90%)`
- Secondary (ochre): `hsl(40 68% 50%)`
- Accent (red): `hsl(352 62% 45%)`

**Rasa visual**
- Lebih “hangat” dan terasa analog.
- Cocok untuk halaman artikel (reading-first).

## Varian C — “Senja Modern”

Inti: merah dibuat lebih muted dan dipasangkan dengan ink slate; accent indigo memberi rasa modern tanpa mematahkan keluarga warna hangat.

**Palet (token → nilai)**
- Paper: `hsl(44 42% 96%)`
- Ink: `hsl(220 16% 14%)`
- Primary: `hsl(350 55% 44%)`
- Primary-weak: `hsl(350 45% 92%)`
- Secondary (ochre): `hsl(36 62% 50%)`
- Accent (indigo): `hsl(227 28% 40%)`

**Rasa visual**
- Lebih “tech/editorial”.
- Bagus untuk landing dengan CTA dan statistik.

## Tipografi (Semua Varian)

- Heading: Anton (uppercase) untuk hierarki kuat dan keterbacaan yang stabil.
- Body: Work Sans untuk teks panjang.
- Meta/label: Space Mono untuk “stempel” editorial.

## Prinsip Komposisi yang Dipakai di Semua Varian

- Pola baca: Z untuk hero/landing, F untuk listing/artikel.
- Kontras minimum: WCAG 2.1 (teks normal ≥ 4.5:1).
- Whitespace: sistem spacing berbasis golden ratio (8–13–21–34–55–89).
- Grid: container konsisten, kolom/spacing stabil di semua halaman.
